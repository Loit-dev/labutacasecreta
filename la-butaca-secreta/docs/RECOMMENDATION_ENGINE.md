# Recommendation Engine

## Objetivo

Encontrar tres recomendaciones que ayuden al usuario a decidir.

No se busca encontrar la película perfecta.

Se busca reducir la indecisión.

---

# Orden de prioridad

1. Disponibilidad en España.
2. Coincidencia con el perfil.
3. Calidad.
4. Variedad.
5. Descubrimiento.

---

# Reglas

Siempre mostrar:

🥇 Mi apuesta para ti

🥈 Otra que creo que disfrutarás

🥉 Una sorpresa que merece una oportunidad

---

# Nunca

- Mostrar contenido no disponible.
- Mostrar tres recomendaciones casi idénticas.
- Repetir constantemente los mismos títulos.

---

# Transparencia

Cada recomendación explicará por qué ha sido elegida.

---

## Principio de Neutralidad

El sistema nunca descartará contenido basándose en suposiciones.

Solo se excluirá una categoría cuando el usuario la haya rechazado explícitamente.

Las respuestas modifican probabilidades, no generan reglas absolutas.
